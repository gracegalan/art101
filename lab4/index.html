Reflection

I will be curious to see the way this new learned skill acts as a building block for the JS we learn moving forward in the course.

  1. initialize the grid with empty cells
  2. set up two players, each with a set of pieces
  3. Main game loop:
    1. display the current state of the grid
    2. ask the current player for their move and chosen column
    3. if the move is valid:
      1. place the player's token in the specified column on the board
      2. check for a win
      3. check for a tie
      4. switch to the next player
  4. display the winner or declare a draw
